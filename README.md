# Katakana Sharp

A focused katakana practice app for Japanese learners who want to improve recognition speed and stop mixing up similar characters.

## Features

- Look-alike character drills with instant feedback
- Current and best streak tracking
- Personal mix-up list, including automatic marking after wrong answers
- Loanword reading practice
- Complete katakana quick-reference chart
- Device-local progress — no account or server required

## Run locally

Requires Node.js 22 or newer.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The static site is written to `out/`. Pushes to `main` deploy automatically to GitHub Pages through the included workflow.
